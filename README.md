دليل سحب الملفات

الوصف

مشروع بسيط لسحب وتنزيل الملفات من مصادر مختلفة عبر الإنترنت.

المميزات

· سحب ملفات من روابط مباشرة
· دعم التنزيل المتعدد
· إظهار تقدم التنزيل
· التعامل مع الأخطاء

المتطلبات

· اتصال بالإنترنت
· Python 3.6 أو أعلى (إذا كان المشروع يعتمد على Python)
· المكتبات المطلوبة (إن وجدت)

طريقة الاستخدام

سحب ملف واحد:

```bash
python downloader.py [رابط الملف]
```

سحب عدة ملفات:

```bash
python downloader.py [رابط1] [رابط2] [رابط3]
```

استخدام قائمة روابط:

1. ضع الروابط في ملف links.txt
2. شغل الأمر:

```bash
python downloader.py -f links.txt
```

الخيارات المتاحة

· -h أو --help: عرض المساعدة
· -o أو --output: تحديد مجلد الحفظ
· -t أو --threads: عدد المسارات المتوازية للتنزيل

أمثلة

```bash
# سحب ملف واحد
python downloader.py https://example.com/file.zip

# سحب ملف مع تحديد مكان الحفظ
python downloader.py https://example.com/file.zip -o ./downloads

# سحب عدة ملفات
python downloader.py https://example.com/file1.zip https://example.com/file2.zip
```

ملاحظات

· تأكد من وجود مساحة كافية على القرص الصلب
· بعض الملفات قد تتطلب صلاحيات خاصة للتنزيل
· احترم حقوق الملكية وقوانين تنزيل المحتوى

الدعم

للمساعدة أو الإبلاغ عن مشاكل، الرجاء فتح issue في صفحة المشروع.
